# Smart Solar Charge Controller Monitoring Dashboard

A real-time IoT-based solar charge controller monitoring system built using **ESP32** and **React + Vite**. The project simulates solar panel and battery parameters, hosts a REST API on the ESP32, and displays live data through an interactive web dashboard with gauges, charts, weather simulation, and motor control.

## Features

- ☀️ Solar Voltage, Current & Power Monitoring
- 🔋 Battery Voltage, SOC & Temperature Monitoring
- 📊 Live Dashboard with Charts & Gauges
- 🌦️ Weather Simulation (Sunny, Cloudy, Rainy)
- 🚗 Motor ON/OFF Control
- 📡 ESP32 REST API Communication
- 📶 Wi-Fi Connectivity
- 🔄 Real-time Data Updates

## Tech Stack

- ESP32 (Arduino)
- React + Vite
- JavaScript
- REST API
- Recharts
- Wi-Fi Communication
